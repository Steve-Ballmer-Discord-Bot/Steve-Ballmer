## Self Host
### Clone the repository
- Open a terminal and type `git clone https://github.com/Steve-Ballmer-Discord-Bot/Steve-Ballmer`
### Enter the folder
- In the same terminal type `cd Steve-Ballmer`
### Install dependencies
- Type `npm i`
### Start the bot
- Type `npm start`
